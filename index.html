Bot Commands

Complete list of available commands for Al Farrizi Music Bot

Music Commands

/play <song> Play a song from YouTube, Spotify, or SoundCloud
/pause Pause the current track
/resume Resume playback
/stop Stop playback and clear the queue
/skip Skip to the next track
/nowplaying Show current playing track info
/seek <time> Seek to a specific time in the track
/volume <1-100> Adjust playback volume

Queue Commands

/queue Display the current queue
/shuffle Shuffle the queue
/loop <track|queue|off> Set loop mode
/remove <position> Remove a track from queue
/clear Clear the entire queue
/move <from> <to> Move a track in the queue

Filter Commands

/filter bassboost <level> Apply bass boost filter
/filter nightcore Apply nightcore effect
/filter vaporwave Apply vaporwave effect
/filter karaoke Remove vocals from track
/filter reset Reset all filters

Admin Commands

/settings View bot settings for this server
/setprefix <prefix> Change the command prefix
/setdj <role> Set DJ role for music commands
/setchannel <channel> Set music command channel
/announce <on|off> Toggle now playing announcements

Frequently Asked Questions

How do I add the bot to my server?

Click the "Invite Bot" button on our website or use the invite link in our Discord server. You'll need "Manage Server" permissions to add the bot.

Why isn't the bot playing music?

Make sure you're in a voice channel, the bot has permissions to join and speak, and there are no region restrictions on the content you're trying to play.

How can I change the bot's prefix?

Server administrators can use the /setprefix command to change the bot's prefix for their server.

Is the bot free to use?

Yes! Al Farrizi Music Bot is completely free to use. We also offer premium features for supporters.
