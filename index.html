Music

Play a song or playlist from various sources

/pause

Pause the current track

/resume

Resume playback

/skip

Skip to the next track

/stop

Stop playback and clear queue

/nowplaying

Show currently playing track

/seek <time>

Seek to a specific position

/volume <1-100>

Adjust playback volume

Queue

/queue

Display the current queue

/shuffle

Shuffle the queue

/loop <off|track|queue>

Set loop mode

/remove <position>

Remove a track from queue

/clear

Clear the entire queue

/move <from> <to>

Move a track in the queue

Click on the invite button on our website or use the /invite command in any server where the bot is present. Make sure you have "Manage Server" permissions.

Make sure you're in a voice channel and the bot has permissions to connect and speak. Also check if the source you're trying to play from is supported.

We support YouTube, Spotify, SoundCloud, Bandcamp, Twitch, Vimeo, and direct HTTP links. Check the Source Managers page for a complete list.
